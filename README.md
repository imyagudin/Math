1. Установите  Mathematica 13.0.0  или  SystemModeler 13.0.0
2. Запустите и выберите  "Other ways to activate"  ->  "Manual Activation"
3. Скопируйте  MathId
4. Запустите  WolframKeygen.exe  и вставьте скопированный  MathId  в самое верхнее поле
5. Нажмите  "Сгенерировать"  чтобы получить  Ключ Активации  и  Пароль Активации
6. Скопируйте их из окна  WolframKeygen  в окно активации в поля  Activation Key  и  Password  соответственно
     Для удобства используйте кнопки копирования рядом с соответствующими полями в  WolframKeygen
7. Не регистрируйтесь online (а также лучше вообще запретите программе выходить в интернет :) )
